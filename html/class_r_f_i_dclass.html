Detailed Description

Main class for the RFID. Initialises an object of MFRC522 which registers if a card is present at the RFID reader or not. Will send an interrupt which triggers a callback to the parkingLot class if a card is present. start() begins a separate thread for every RFID reader and stop() ends it. registerCallback needs to be called from host class (parkingLot) to be able to get data.
