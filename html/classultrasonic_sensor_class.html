Detailed Description

Main class for the ultrasonic sensors. Registers changes from the input GPIO pin and sends an interrupt which triggers a callback to the parkingLot class if the distance is in the range of change for a car leaving or arriving. start() begins a separate thread for every ultrasonic sensor and stop() ends it. registerCallback needs to be called from host class (parkingLot) to be able to get data.
