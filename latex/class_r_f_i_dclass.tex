\doxysubsection{Detailed Description}
Main class for the RFID. Initialises an object of \mbox{\hyperlink{class_m_f_r_c522}{MFRC522}} which registers if a card is present at the RFID reader or not. Will send an interrupt which triggers a callback to the \mbox{\hyperlink{classparking_lot}{parking\+Lot}} class if a card is present. \mbox{\hyperlink{class_r_f_i_dclass_a233dde423fc0077b461cfea59f27f440}{start()}} begins a separate thread for every RFID reader and \mbox{\hyperlink{class_r_f_i_dclass_aedd32f30cfc48d1a551173672d85a4b4}{stop()}} ends it. register\+Callback needs to be called from host class (\mbox{\hyperlink{classparking_lot}{parking\+Lot}}) to be able to get data.
