\doxysubsection{Detailed Description}
Main class for the \mbox{\hyperlink{class_buzzer_class}{Buzzer\+Class}}. Informs car that has parked in wrong spot that it should move. Buzzer turns on if car is in wrong spot. Buzzer turns off when car moves away.
