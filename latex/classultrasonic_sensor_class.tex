\doxysubsection{Detailed Description}
Main class for the ultrasonic sensors. Registers changes from the input GPIO pin and sends an interrupt which triggers a callback to the \mbox{\hyperlink{classparking_lot}{parking\+Lot}} class if the distance is in the range of change for a car leaving or arriving. \mbox{\hyperlink{classultrasonic_sensor_class_a73df157512f05550c4c69a0733a910e1}{start()}} begins a separate thread for every ultrasonic sensor and \mbox{\hyperlink{classultrasonic_sensor_class_ad13c697cd085af6c0c6a6f3c020fa972}{stop()}} ends it. register\+Callback needs to be called from host class (\mbox{\hyperlink{classparking_lot}{parking\+Lot}}) to be able to get data.
